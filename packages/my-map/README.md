# my-map #

Easily embed a map powered by [Leaflet][].

* [nfreear.github.io/elements/demo][demo]

## Usage

Available on the [Unpkg][] CDN:

```html
<my-map></my-map>

<script src="https://unpkg.com/my-map@1" type="module" ></script>
```
