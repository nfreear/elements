Load a WOFF font file, and make it available for use in CSS styles.

On this page we're loading several fonts using Fontsource:

  • OpenDyslexic font — a typeface designed against some common symptoms of dyslexia.
  • Ma Shan Zheng — reminiscent of fonts used to display "yinglian," the short poems and blessings traditionally posted on either side of the entryway to a home or temple.
  • Material Icons — An icon font.
